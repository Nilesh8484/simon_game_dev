# 🧠 Simon Game

A fun and interactive **Simon Memory Game** built using **HTML, CSS, and JavaScript**. The game tests your memory by generating a sequence of colors that you must repeat correctly. As you progress, the sequence becomes longer and more challenging.

---

## 📖 About

The Simon Game is inspired by the classic electronic memory game where players repeat an increasingly long sequence of colors and sounds.

This project helped me improve my understanding of:

- DOM Manipulation
- Event Handling
- JavaScript Logic
- Arrays
- Game Development Basics
- CSS Animations
- Audio Integration

## 🎮 How to Play

1. Press any key (or Start button).
2. Watch the color sequence.
3. Repeat the sequence by clicking the buttons.
4. Each level adds one new color.
5. Continue as long as you remember the pattern.
6. If you click the wrong button, the game ends.
7. Press any key to restart.

---

## 🎯 Game Rules

- ✅ Repeat the sequence exactly.
- ✅ The sequence grows after every level.
- ❌ One wrong click ends the game.
- 🏆 Try to achieve the highest level possible.
